關於我們的實驗室

訊號處理實驗室著重於將前沿的數學分析工具，應用於機電系統與生物醫學工程。我們致力於發展高精度、高強健性的訊號與影像分析演算法。

核心研究領域

📊

經驗模態分析 (EMD)

Empirical Mode Decomposition

專注於非線性與非穩態訊號處理技術，發展改進型的 EMD 演算法（如高斯窗口 EMD、適應性時間-尺度分解），解決傳統分析工具在時頻解析度與混波現象上的限制。

🔍

獨立成分分析 (ICA)

Independent Component Analysis

利用統計獨立性將混合訊號分離成原始訊號。應用於盲源訊號分離、生醫訊號降噪以及多管道感測器資訊特徵提取。

⚙️

錯誤診斷系統

Fault Diagnosis System

結合多尺度排序熵 (MPE)、支援向量機 (SVM) 與馬氏距離，開發針對旋轉機械（齒輪箱、滾珠軸承）的即時異常檢測與劣化預警系統，實現智慧工廠之預防性保養。

❤️

生醫訊號分析

Biomedical Signal Analysis

針對心電圖 (ECG)、腦波圖 (EEG) 及肌電圖 (EMG) 等生理訊號，開發高效降噪（濾除電極接觸雜訊與工頻干擾）與波形特徵自動辨識演算法，提升臨床診斷精度。

👁️

生醫影像分析

Biomedical Image Analysis

研發超頻譜生醫影像擷取與快速分析系統。例如利用「繼光鏡掃描式超頻譜影像系統」進行大鼠聲門與口腔癌早期診斷的影像偵測與病變分割演算法。

實驗室設備與硬體資源

為了支援學生進行具產業深度的實務研究，本實驗室配置了豐富的量測儀器、硬體開發板以及運算伺服器：

📊 量測儀器與硬體設備

  • 高階數位儲存示波器：提供多通道、高取樣率的動態訊號擷取與量測。
  • 數位精密函數訊號產生器：用於產生標準及任意波形的測試訊號。
  • DSP 數位訊號處理器開發套件：Texas Instruments (TI) 高階 DSP 開發板，供演算法硬體加速實驗。
  • 多通道生醫生理訊號擷取模組：具備高度隔離降噪電路，用於 ECG/EMG 實際訊號量測。

💻 軟體環境與運算伺服器

  • 高效能 GPU 運算工作站：配備多張 NVIDIA 高階顯卡，用於執行深度學習與大型訊號分析模型訓練。
  • MATLAB & Simulink：提供完整的訊號處理、小波轉換、控制系統模擬之開發授權。
  • LabVIEW 虛擬儀器開發平台：用於電腦與硬體儀器的資料擷取 (DAQ) 及圖形化監控界面整合。
  • Python 機器學習環境：配置 PyTorch / TensorFlow 開發框架，專注於智慧化錯誤診斷演算法。
