我們的核心技術與研究方向

我們將領先的時頻分析理論，轉化為具備商業落地價值的低功耗智慧地下物聯網（AIoUT）解決方案。以學術為深度，以產業為廣度。

衍生企業「譽科科技」三大核心技術

本實驗室技術衍生創辦之「譽科科技股份有限公司」專注於高風險與極端環境，提供從感測端、傳輸端到平台端的一站式智慧監測系統。

🔋

極低功耗感測設計

Low-Power Sensor Design

自主研發低功耗雷達波水位計與無線通訊記錄器，功耗僅為傳統設備的 1/5，平均每筆量測消耗電流低至 1.9mA，大幅延長野外設備的電池續航，降低維護成本。

🧠

AI 邊緣運算與智能辨識

Edge AI & Intelligent Recognition

將機器學習演算法布建於邊緣端硬體，實作人孔蓋巡檢、管線異常辨識、水色辨識與職安工安監控。透過高效邊緣運算，將後端平台資料處理與顯示速度提升 5 倍。

📡

長距離無線傳輸架構

Long-Range Wireless Architecture

支援 WiFi HaLow、4G LTE Cat1 與低軌衛星通訊。利用 WiFi HaLow 穿透性強、傳輸距離遠（達 1 公里）的特性，不論是繁雜都市叢林地下道，或是偏遠山區，皆能確保數據穩定回傳。

實驗室核心學術研究領域

譽科科技的核心技術源自實驗室於數位訊號處理、時頻分析與生醫工程的深厚學術積累：

📊

經驗模態分析 (EMD)

Empirical Mode Decomposition

專注於非線性與非穩態訊號處理技術，發展改進型的 EMD 演算法（如高斯窗口 EMD、適應性時間-尺度分解），解決傳統分析工具在時頻解析度與混波現象上的限制。

🔍

獨立成分分析 (ICA)

Independent Component Analysis

利用統計獨立性將混合訊號分離成原始訊號。應用於盲源訊號分離、生醫訊號降噪以及多管道感測器資訊特徵提取。

⚙️

錯誤診斷系統

Fault Diagnosis System

結合多尺度排序熵 (MPE)、支援向量機 (SVM) 與馬氏距離，開發針對旋轉機械（齒輪箱、滾珠軸承）的即時異常檢測與劣化預警系統，實現智慧工廠之預防性保養。

❤️

生醫訊號分析

Biomedical Signal Analysis

針對心電圖 (ECG)、腦波圖 (EEG) 及肌電圖 (EMG) 等生理訊號，開發高效降噪（濾除電極接觸雜訊與工頻干擾）與波形特徵自動辨識演算法，提升臨床診斷精度。

👁️

生醫影像分析

Biomedical Image Analysis

研發超頻譜生醫影像擷取與快速分析系統。例如利用「繼光鏡掃描式超頻譜影像系統」進行大鼠聲門與口腔癌早期診斷的影像偵測與病變分割演算法。

實驗與研發設備

為了支援學生進行具產業深度的實務研究，本實驗室與譽科科技配置了豐富的量測儀器、硬體開發板以及運算伺服器：

📊 量測儀器與物聯網套件

  • 高階數位儲存示波器：提供多通道、高取樣率的動態訊號擷取與量測。
  • 低功耗通訊測試儀：用於 WiFi HaLow 等長距離低功耗通訊的發射功率與強健性分析。
  • DSP 數位訊號處理器開發套件：Texas Instruments (TI) 高階 DSP 開發套件，供演算法硬體加速實驗。
  • 多通道生醫生理訊號擷取模組：具備高度隔離降噪電路，用於 ECG/EMG 實際訊號量測。

💻 軟體環境與運算伺服器

  • 高效能 GPU 運算工作站：配備多張 NVIDIA 高階顯卡，用於執行邊緣 AI 辨識模型之大型深度學習模型訓練。
  • MATLAB & Simulink：提供完整的訊號處理、小波轉換、控制系統模擬之開發授權。
  • LabVIEW 虛擬儀器開發平台：用於電腦與硬體儀器的資料擷取 (DAQ) 及圖形化監控界面整合。
  • Python 機器學習與環境：配置 PyTorch / TensorFlow 開發框架，專注於智慧化錯誤診斷與管線異常分析。
