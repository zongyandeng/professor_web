🎯 實驗室培訓特色與重點

我們實驗室非常著重同學的核心動手能力與問題解決能力。在學期間，您將有機會參與並學習以下核心技能：

  • 演算法開發與驗證：熟練運用 MATLAB 與 Python 處理真實世界的非線性、非穩態訊號。
  • 硬體與嵌入式開發：學習微處理器電路設計、數位訊號處理器 (DSP) 與 ARM 架構的韌體撰寫。
  • 人工智慧應用：將機器學習與深度學習應用於故障診斷與生醫辨識。
  • 論文發表與簡報訓練：每週固定的論文研討與研究報告，培養邏輯表達與學術寫作能力。

✨ 期望招募的學生特質

我們歡迎符合以下特質的同學加入我們的研究團隊：

  • 主動積極的學習態度：遇到研究瓶頸時願意嘗試尋找解決方案，主動與教授及學長姐討論。
  • 扎實的程式或工程基礎：具備 C/C++、MATLAB 或 Python 基礎者佳，或對韌硬體整合有極大興趣。
  • 良好的團隊合作精神：樂於分享研究心得，並能與實驗室成員共同維護良好的研究氣氛。
  • 歡迎專題生提早加入：歡迎大二下、大三同學前來諮詢專題研究，提早進入實驗室進行基礎技能培訓。
